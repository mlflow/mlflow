# Cookbooks

Practical, hands-on examples for building GenAI applications with MLflow. Each cookbook is a self-contained guide with working code.

## Evaluation

- [End-to-End RAG Evaluation](/genai/cookbooks/rag-evaluation) — Build a RAG pipeline and evaluate retrieval + generation quality with built-in judges.
- [Building Custom LLM Judges](/genai/cookbooks/custom-llm-judges) — Create domain-specific scorers beyond the built-in judges.
- [Evaluation-Driven Development](/genai/cookbooks/eval-driven-development) — Use evaluation as the driver for iterative improvement of a GenAI app.
- [Cost-Quality Tradeoff Analysis](/genai/cookbooks/cost-quality-tradeoff) — Compare quality and cost across LLM providers to find the right balance.
- [Red-Teaming Your LLM Application](/genai/cookbooks/red-teaming) — Test your app against adversarial inputs with safety scorers.

## Agents

- [Evaluating a Multi-Turn Conversational Agent](/genai/cookbooks/multi-turn-agent) — Evaluate multi-turn chat quality with conversational scorers.
- [Tracing and Evaluating a LangGraph Agent](/genai/cookbooks/langgraph-agent) — Instrument a LangGraph agent and evaluate tool calling accuracy.
- [Tracing and Evaluating OpenAI Agents](/genai/cookbooks/openai-agents) — Trace OpenAI function-calling agents and evaluate end-to-end.

## Prompt Engineering

- [Prompt Engineering Lifecycle](/genai/cookbooks/prompt-engineering) — Version, test, and optimize prompts with MLflow's prompt registry.

## Production

- [Production Observability](/genai/cookbooks/production-observability) — Set up tracing, monitoring, and alerting for production GenAI apps.

## Databricks Genie

- [Genie Conversation Tracing Pipeline](/genai/cookbooks/genie-tracing-pipeline) — Ingest Genie conversations into Delta and trace each interaction with MLflow.
- [Genie Evaluation with LLM Judges](/genai/cookbooks/genie-evaluation-judges) — Run automated judges and human labeling on Genie traces.
- [Genie Space Analyzer](/genai/cookbooks/genie-space-analyzer) — Audit your Genie space and generate configuration recommendations.
- [Genie Vector Search and Lakebase](/genai/cookbooks/genie-vector-search) — Add semantic search over past Genie conversations.
