# Coding Agents & AI Gateway

The MLflow AI Gateway supports popular subscription-based AI coding assistants — **Claude Code**, **OpenAI Codex**, and **Gemini CLI** — out of the box.

When these tools send requests through the gateway, it automatically detects them via their `User-Agent` header and forwards their own credentials to the upstream provider, instead of substituting the server-side API key. This means:

- Users authenticate with their **personal or team subscription**
- The gateway still provides **centralized observability and tracing**
- No changes are needed to the CLI tool configuration beyond pointing it at the gateway

:::info
This passthrough behavior only activates when the CLI includes its own auth header (`Authorization`, `x-api-key`, or `x-goog-api-key`). If no client credential is present, the gateway falls back to the server-side API key as usual.
:::
