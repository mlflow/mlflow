# Coding Agents & AI Gateway

The MLflow AI Gateway supports popular subscription-based AI coding agents, **Claude Code**, **OpenAI Codex**, and **Gemini CLI**, out of the box.

Routing coding agents through the gateway gives your team two key capabilities:

## Observability

Every request made by a coding agent is automatically captured as an MLflow trace, giving you full visibility into what your team's agents are doing — inputs, outputs, model used, token counts, and latency — without any code changes.

[→ Learn about Usage Tracking](/genai/governance/ai-gateway/usage-tracking)

## Budget Control

Set spending limits per endpoint to prevent runaway costs from coding agent sessions. Configure alerts to notify you before a budget is exhausted, and hard limits to stop requests once the threshold is reached.

[→ Learn about Budget Alerts & Limits](/genai/governance/ai-gateway/budget-alerts-limits)

---

Configuration is minimal, just set an environment variable.
